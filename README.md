## Setup

### Requirements

- Node and NPM
- Ionic CLI

### Installation

1.  Clone the repo `$ git clone git@github.com:calebfroese/ivy-ionic.git`
1.  Install packages `$ npm install`
1.  Serve the app `$ npm run ionic:serve`
